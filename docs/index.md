# 42 CPP Modules Guide

!!! info "このサイトについて"
    42 の **cpp00〜cpp04** を題材に、**C 経験者が C++ を最短で理解する**ための解説サイトです。
    各 exercise について、**仕様・実行例・コード解説・テストチェックリスト・評価対策**を統一フォーマットで掲載しています。

## このガイドの読み方

=== "C++ 初めて"
    以下の順で読むのがおすすめです。

    1. **[C→C++入門](intro/c-vs-cpp.md)** — C との違いの核心 (15分)
    2. **[開発環境](intro/setup.md)** — コンパイラと必須フラグ
    3. **[cpp00 概要](cpp00/index.md)** → ex00 → ex01 → ex02 の順で進む
    4. 詰まったら **[対応表](intro/cheatsheet.md)** を常時参照

=== "復習したい"
    必要な箇所にピンポイントで飛んでください。

    - [用語集](reference/glossary.md) — C++ 用語をC言語ベースで説明
    - [提出前チェック](reference/submit-check.md) — 評価で落ちやすいポイント
    - 右上の🔍検索は日本語対応

## 各 exercise ページの構成

すべての exercise ページは以下 8 セクションで統一されています。

| # | セクション | 目的 |
|---|---|---|
| 1 | このexerciseで学ぶこと | 3行で概要 |
| 2 | 課題仕様 | subject の日本語化 |
| 3 | 実行例 | 実際のターミナル記録 |
| 4 | C → C++ の新概念 | C経験者のためのブリッジ |
| 5 | コード解説 | 機能単位のウォークスルー |
| 6 | テストチェックリスト | 評価観点を網羅 |
| 7 | よくある落とし穴 | 禁止事項と典型ミス |
| 8 | 発展キーワード | 次に学ぶこと |
